Important to consider

Keep these rules simple and strict.

DWG is the default

Use Workflow A unless you specifically want the grade to live inside ARRI-style color science.

No CST before ColorClone

Workflow B starts with ColorClone. Adding a CST before it is unnecessary and weakens the logic of the chain.

LogC4 needs 10000 nits

The LogC4 output CST should use Max Input Nits 10000, not 1000.

Measure late, correct early

Broadcast Safe and measurement happen after the Output CST. Read late, but make corrections earlier in the chain.

Shared rules

What stays the same in both workflows.

Same discipline

Both workflows follow the same serial node logic and the same grading order.

Same grading zone

Do the creative work in the middle of the chain, not on the infrastructure nodes.

Same output logic

Both workflows end in Rec.709, so delivery stays consistent once the Output CST is correct.

Same mindset

Keep the structure clean: one operation per node, clear intent, no unnecessary transforms.
